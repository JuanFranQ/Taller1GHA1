Implementación de Integración y Despliegue Continuo

¿Qué es CI/CD?

CI/CD significa Integración Continua y Despliegue Continuo. Es una práctica de desarrollo de software que permite:

  • Automatizar el proceso de integración de código
  • Realizar pruebas de forma automática
  • Desplegar aplicaciones de manera consistente

GitHub Actions

GitHub Actions es una plataforma que nos permite:

  • Automatizar flujos de trabajo
  • Ejecutar pruebas automáticas
  • Realizar despliegues automáticos
